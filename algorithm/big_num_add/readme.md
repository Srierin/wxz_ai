- es6 新增了第六种简单数据类型BigInt类型，用于表示大整数


## BigInt
   安全 2^53 - 1 = 9007199254740991
   在后面加n 表示大整数
   可以使用BigInt(“ ”) 函数将普通数字转换为大整数 不能new
   无限大 无溢出问题
   js 适合大型项目开发
   BigInt(“9007199254740991”) + BigInt(“1”) 会报错  
   可以使用BigInt(“9007199254740991”) + 1n 不会报错
